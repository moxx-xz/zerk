## 📖 Descripción

**Zerk** es una Single Page Application (SPA) que unifica en un solo entorno web las tres experiencias que los jugadores más demandan:

| Referente | Funcionalidad adoptada en Zerk |
|-----------|-------------------------------|
| 🎮 Steam | Tienda de videojuegos: compra, descarga, biblioteca personal, historial de precios y descuentos |
| 💬 Discord | Canales de voz y texto, servidores de comunidad por juego, mensajería directa |
| 🐦 Twitter / Foros | Feed de reseñas, publicaciones de opinión, hilos de discusión, sistema de reputación |

La propuesta nace de una necesidad concreta: los jugadores modernos deben mantener abiertas múltiples pestañas o aplicaciones simultáneamente, lo que genera fragmentación de la experiencia y consumo innecesario de recursos.

---

## 🧩 Módulos Principales

La navegación se organiza a través de un **menú de hamburguesa lateral** que agrupa todas las funcionalidades:

1. **Inicio / Explorar** — Feed dinámico con juegos destacados, actividad de amigos y tendencias
2. **Tienda** — Catálogo con filtros de género, precio, descuentos y valoración
3. **Biblioteca** — Juegos adquiridos, organizados por estado (jugando, pendiente, completado)
4. **Comunidad / Foro** — Foros por juego, hilos de discusión y reseñas
5. **Chat / Canales** — Canales de texto y voz por servidor; mensajería directa
6. **Comparador de Precios** — Compara el precio de un juego en múltiples tiendas digitales
7. **Perfil** — Estadísticas, logros, amigos y publicaciones del usuario
8. **Notificaciones** — Alertas de descuentos, respuestas en foros y mensajes nuevos
9. **Configuración** — Preferencias de cuenta, privacidad y apariencia

## 🛠️ Tecnologías Previstas

| Capa | Tecnología | Función |
|------|-----------|---------|
| Frontend | HTML5, CSS3, JavaScript | Estructura, estilo e interactividad |
| Framework UI | React.js o Vue.js | Componentes reutilizables y SPA |
| Backend | Node.js + Express o Django | Lógica de negocio y APIs REST |
| Base de datos | PostgreSQL o MySQL | Almacenamiento de usuarios, juegos y mensajes |
| Tiempo real | WebSockets (Socket.io) | Chat en vivo y notificaciones instantáneas |
| Autenticación | JWT + OAuth 2.0 | Inicio de sesión seguro y vinculación con terceros |
| Control de versiones | Git + GitHub | Gestión del código fuente y trabajo colaborativo |

## 🔄 Flujo de Usuario Principal

```
Registro / Login
      ↓
Inicio — feed de juegos populares y publicaciones
      ↓
Tienda — filtrar, comparar precios y comprar
      ↓
Comunidad — leer reseñas y publicar opiniones
      ↓
Chat — unirse a servidores y conversar en tiempo real
      ↓
Perfil — completar datos, agregar amigos y personalizar
```
